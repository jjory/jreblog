# 일본 부동산 → 네이버 블로그 자동화 시스템 제안서

## 🎯 비즈니스 목표
한국 손님의 모객 증가를 위해, **일본 부동산 마이소크(物件図面)를 입력하면 한국인 관점에 최적화된 네이버 블로그 글이 자동 생성·발행**되는 파이프라인 구축.


### 왜 Claude Opus 4.7 Vision인가?
- **3배 향상된 이미지 해상도 지원** (2,576px / 3.75MP) — 마이소크의 작은 글씨도 정확히 인식
- **시각 정확도 98.5%** (XBOW 벤치마크) — 4.6 대비 약 2배 정확도
- 일본어 OCR + 도면 구조 이해를 한 번에 처리
- 평면도(間取り)를 보고 채광·동선·실용성까지 분석 가능

## 🎨 한국인 맞춤 블로그 콘텐츠 전략

### 1. SEO 제목 공식
```
[지역명][역명] 1LDK 월세 ○○만엔 │ [핵심 장점] - 외국인 OK
예: [신주쿠/오쿠보역] 1K 7.5만엔 │ 한인타운 도보 5분 - 워홀러 추천
```

### 2. 본문 구조 (네이버 블로그 SEO 최적화)
1. **Hook (첫 200자)**: 핵심 가격·위치·타깃 비자 (검색 노출 핵심)
2. **물건 기본정보** (표 형태)
3. **초기비용 총정리** ⭐ — 한국인이 가장 궁금해하는 부분
   - 일본 제도 설명 + 한국 원화 환산 (100엔 ≈ 950원)
4. **위치·생활 인프라**
   - 한인 마트, 한국 음식점, 공항 접근성
5. **비자별 적합도 매칭**
   - 워홀: 단기계약 가능 여부
   - 유학: 학교 통학 시간
   - 취업: 주요 비즈니스 구역 접근성
   - 경영관리: 사무실 겸용 가능 여부
6. **계약 진행 절차** (외국인 특화)
7. **CTA**: 카카오톡/라인 상담 링크
8. **해시태그**: `#도쿄월세 #신주쿠원룸 #일본워홀집` 등

## 🔐 네이버 블로그 발행 — 두 가지 옵션

### 옵션 A: 완전 자동화 (Naver OpenAPI)
- **엔드포인트**: `https://openapi.naver.com/blog/writePost.json`
- **요구사항**:
  1. 네이버 개발자 센터 앱 등록
  2. **네아로(네이버 아이디로 로그인) 심사 통과 — 약 3일 소요**
  3. OAuth 2.0 access_token 획득
- **장점**: 완전 자동화, 대량 처리
- **주의**: 도배성 글 자동발행은 노출 저하 위험 → 발행 간격 조절 필요

### 옵션 B: SmartEditor HTML 출력 (즉시 사용 가능)
- 네이버 블로그 글쓰기 창에 **복붙만 하면 되는 HTML** 생성
- 심사 없이 바로 사용
- **추천**: 초기에는 옵션 B로 운영 → 충분히 검증되면 옵션 A 추가

본 시스템은 **두 옵션 모두 지원**합니다.

## 🛠 기술 스택

- **언어**: Python 3.10+
- **AI**: Claude Opus 4.7 (Vision + Text)
- **UI**: Streamlit (사내 운영자용 웹 UI)
- **발행**: Naver OpenAPI (OAuth 2.0)
- **배포**: AWS Lambda + S3 또는 사내 PC 구동

## 🚀 향후 확장 가능성

- **영어/중국어/대만어 동시 생성** → 자사 영문 블로그·小紅書·Facebook 동시 발행
- **시세 비교 자동 인서트** — 같은 지역 유사 물건 시세 자동 표시
- **카카오톡 챗봇 연동** — 손님이 키워드 입력 시 매칭 매물 자동 추천
- **LINE 공식계정** — 일본 거주 한국인 대상 신착 매물 자동 알림

# 📝 스타일 프리셋 폴더

이 폴더에 있는 `.md` 파일들은 블로그 글의 톤·구조를 결정합니다.

## 사용법

1. **앱 사이드바의 "글 스타일" 드롭다운**에서 사용할 스타일 선택
2. 블로그 글 생성 시 해당 스타일로 작성됨

## 새 스타일 추가하기

1. 이 폴더에 `새스타일이름.md` 파일 생성
2. 톤, 인사말 예시, 강조 사항을 작성 (다른 파일 참고)
3. 앱을 재시작하면 드롭다운에 자동으로 나타남

## 기존 스타일 수정하기

각 `.md` 파일을 VSCode나 메모장으로 열어 수정 후 저장.
다음 블로그 글 생성 시부터 즉시 반영됩니다.

## 주의사항

- 파일명에 특수문자·공백 사용 자제 (한글은 OK)
- `README.md`와 `_`로 시작하는 파일은 무시됨
- 사실 정확성·외국인 정보 명확성은 어떤 스타일에서도 유지되어야 합니다 (시스템 기본값)
